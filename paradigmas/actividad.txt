--> REALIZAR UN SIMPLE MENU DE 4 OPCIONES :
   1. CREAR
   2. ACTUALIZAR
   3. BORRAR
   4. SALIR

========================================== FUNCIONAMIENTO ======================================
1. SE INICIALICE UNA VARIABLE CON UNA OPCION, ESTA VARIABLE DEBE SER DE TIPO INT
2. SE MUESTRE UN TITULO "BIENVENIDO A MI MENU SIMPLE" (UNICA VEZ)
3. SE MUESTRE LAS 4 OPCIONES ANTERIORMENTE MENCIONADAS
4. SI LA VARIABLE QUE CONTIENE LA OPCION NO ENTRA EN ALGUNA DE LAS OPCIONES :
    4 a. MOSTRAR DE NUEVO LAS OPCIONES DEL PASO 3
    4 b. AUMENTAR EN 1 LA OPCION (EN CASO DE QUE ESTA SEA MENOR A LA PRIMERA OPCION POSIBLE) O DESCONTAR EN 1 (EN CASO DE QUE ESTE SEA MAYOR A LA ULTIMA OPCION POSIBLE)
5. SI LA VARIABLE MATCHEA CON ALGUNA DE LAS OPCIONES ANTERIORES, MOSTRAR EL SIGUIENTE MENSAJE " ELEGISTE LA OPCION {VALOR DE LA OPCION} " Y FINALIZAR EL PROGRAMA.
